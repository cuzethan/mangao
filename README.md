## IMPORTANT

Make sure to make a .env fine inside your "backend/database" folder

ex.

> POSTGRES_DB=mangao_db
>
> POSTGRES_USER=mangao_user
>
> POSTGRES_PASSWORD=mangao_password
>
> PGUSER=mangao_user
>
> PORT = 3000
>
> PGPASSWORD=mangao_password
> 
> PGDATABASE=mangao_db
>
> PGHOST=db
>
> PGPORT=5432
